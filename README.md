# Kodexa Homebrew Tap

**Copyright © 2026 Kodexa, Inc. All rights reserved.**

Official Homebrew tap for Kodexa CLI tools.

## Installation

```bash
# Add the tap
brew tap kodexa-ai/tap

# Install kdx CLI
brew install kdx

# Upgrade kdx
brew upgrade kdx
```

## Available Formulae

- **kdx** - Kodexa CLI for managing Kodexa AI Platform resources

## About

This tap provides easy installation of Kodexa command-line tools on macOS and Linux systems.

For more information, visit [kodexa.ai](https://kodexa.ai).
